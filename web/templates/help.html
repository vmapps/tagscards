  • how to export contacts as JSON file ?

    • - Click " JSON export" button in header bar.

  • how to export users as JSON file ?

    • - Once logged in, click "users" button in header bar.
    - Then click button below list of users.

  • how to import contacts from CSV file ?

    • - Once logged in, click " CSV import" button in header bar.
    - Then choose CSV to import and click "import data" button.
    - CSV file must contain one contact per line.
    - Each line must contain fields must be separated with semicolons.
    - Required fields are "fullname;role;email;pgp;phone;website;tags".
    - Tags must be separated with commas "tag1,tag2,tag3".
